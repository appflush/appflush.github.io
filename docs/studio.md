App Studio
===================

이 문서에서는 App Studo의 인터페이스를 설명 합니다.

App Studio는 사용자가 App을 제작하는 공간 입니다. App을 제작하는데 필요한 모든 도구들을 갖추고 있으며 그 밖에도 다양한 확장 유틸리티들을 포함하고 있습니다. Graphical User Interface를 구축해 접근성을 높였으며 Device로그를 실시간으로 확인 하실 수 있습니다.
