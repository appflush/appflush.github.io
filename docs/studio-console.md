App Studio - Console
===================

App Studio Console 기능은 App Studio 및 디바이스에서 보내는 로그를 받아 출력하는 역할을 합니다. 여기에는 사용자가 App을 제작하는 과정 중에 발생하는 일반적인 에러 메세지도 포함 합니다.
